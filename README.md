# ARCADIA

ARCADIA is a single-file static browser games hub. The site entry point is `index.html`.

## Publish On GitHub Pages

Upload these files and folders to the root of the `SunsetTORCH.github.io` repository:

- `index.html`
- `404.html`
- `.nojekyll`
- `.github/workflows/pages.yml`

Then open the repository on GitHub and go to `Settings > Pages`.

Use either option:

- Recommended with this workflow: set `Source` to `GitHub Actions`.
- Simple branch publishing: set `Source` to `Deploy from a branch`, `Branch` to `main`, and `Folder` to `/ (root)`.

After GitHub finishes deploying, the site should be available at:

`https://SunsetTORCH.github.io`
